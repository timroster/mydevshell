# Notes on settings things up

## permissions for users

By default minishift appears not to include cluster-admin access for the admin user. This has to be added separately using:

```
oc login -u system:admin
oc adm policy add-cluster-role-to-user cluster-admin admin
```

Once complete, the admin user can login and see security contexts

oc get scc
